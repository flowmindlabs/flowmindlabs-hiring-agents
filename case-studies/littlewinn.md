# Case Study — LittleWinn

## The Problem

1 in 8 children in India has a developmental challenge — autism, ADHD, Down syndrome, developmental delays, speech delays, or learning disabilities.

Their parents struggle daily with:
- Managing daily routines and schedules
- Understanding and responding to behavior challenges
- Teaching life skills at home
- Finding structured, affordable support in Hindi

**Almost no digital tools exist for Indian special needs families.**
No Hindi support. Nothing affordable. Nothing built for India.

## What FlowMind Labs Built

**LittleWinn** — a full-stack AI-powered Progressive Web App (PWA) for parents of children with special needs.

### Features Delivered
- **AI Behavior Guide** — explains why a child behaves a certain way and tells parents exactly what to do
- **Daily Schedule Manager** — customizable daily routine with push notification reminders
- **Life Skills Training** — step-by-step guides with AI-generated images and audio narration
- **Child Assessment** — tracks developmental progress over time
- **Multi-child Profile Management** — manage up to 5 children in one account
- **Admin Dashboard** — real-time monitoring and user management
- **Public Status Page** — live service health monitoring
- **Full Hindi Language Support** — complete UI and content in Hindi
- **Push Notifications** — via Firebase Cloud Messaging

### Technology Stack
- **Frontend:** React, Tailwind CSS, Shadcn UI, React Portals
- **Backend:** FastAPI (modular architecture), Python
- **Database:** MongoDB
- **AI:** Anthropic Claude, Google Gemini, OpenAI TTS
- **Notifications:** Firebase Cloud Messaging
- **Email:** Resend
- **Languages:** English + Hindi (Tamil, Telugu, Kannada via Sarvam AI — coming soon)

### Architecture
Refactored from monolithic to modular:
- 13 route files
- Clean separation of models, routes, services, data
- Secure JWT authentication
- React Portals for UI stability

## Results

- **Complete production-ready PWA** — no app store download needed
- **Works on any phone** — add to home screen from browser
